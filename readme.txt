Awesome breadcrumb for frontend plugin for visitors keep track thier location and BFF(Breadcrumb For Frontend) plugin is fully compatiable with all devices.

== Description ==

Breadcrumb For Frontend is awesome feature for visitors keep track thier location, by this plugin you can display breadcrumb navigation anywhere your website via short-codes. Breadcrumb For Frontenf Plugin is used to indicate the current page’s location within a navigational hierarchy.

Major features in Breadcrumb For Frontend include:

* You can use it via short-codes.
* Simple, easy to use interface – perfect for individual users & clients.
* On dasboard plugin email support link available
* Custom text in-front of Breadcrumb For Frontend.
* Breadcrumb For Frontend is a fully responsive plugin.
* Developer friendly plugin.
* Breadcrumb display on any post type, page, parent pages, author page, archive page, tag page, custom post type, custom taxonomy page, search page, 404 page. 


== Installation ==

1. Install as regular WordPress plugin.
2. Go your plugin setting via WordPress Dashboard and find "Breadcrumb For Frontend" activate it.

== Frequently Asked Questions ==

= Why was this plugin created? =

= Do you have any shortcode to display breadcrumb in frontend ? =
Yes

= How to display breadcrumb in frontend ? =
You can use this short-code any where to display breadcrumb
'[bcff_display_breadcrumb]'

= Will Breadcrumb For Frontend work with my theme? =
Yes; Breadcrumb For Frontend will work with any theme.

= After install this plugin in admin dashbaord any settings options available? =
No

= Do breadcrumbs show in Google search results? =
No
